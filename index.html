  • HTTP Device API - easy API for reading from and writing to One Platform dataport resources via HTTPS
  • CoAP Device API - small, low bandwidth protocol for low-power devices and lossy networks
  • Remote Procedure Call (JSON RPC) API - full featured API for device management and data visualization via HTTPS
  • Remote Procedure Call API via WebSocket - RPC API over websocket with support for subscriptions

Portals Web Application

Portals is a whitelabel dashboard based web application framework that is used for building production applications, proof of concepts, and evaluations of Exosite's platform. Portals has a concept of user identity and permissions and strictly uses the One Platform hierarchy for device ownership control.
