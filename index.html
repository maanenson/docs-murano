<h1 id="documentation-overview">Documentation Overview</h1> <h3 id="tutorials">Tutorials</h3> <ul> <li><a href="tutorials/get-started">Getting Started</a> - Using a free account, get a device added and start sending data</li> <li><a href="videos/">Videos</a> - Video tutorials covering various topics</li> </ul> <h3 id="device-apis">Device APIs</h3> <ul> <li><a href="coap/">CoAP API</a> - small, low bandwidth protocol for low-power devices and lossy networks</li> <li><a href="http/">HTTP API</a> - easy API for reading from and writing to dataports</li> <li><a href="udp/">UDP Single Shot API</a> - very low bandwidth and overhead API only for writing to dataports</li> </ul> <h3 id="application-apis">Application APIs</h3> <ul> <li><a href="rpc/">Remote Procedure Call API</a> - full featured API for device management and data visualization</li> <li><a href="provision/">Provision Management API</a> - API for working with device models and in field provisioning</li> <li><a href="commander/">Commander API</a> - XMPP chat interface for reading and writing data from an IM client</li> <li><a href="scripting/">Lua Scripting API</a> - the server-side Lua scripting environment</li> <li><a href="widget/">Custom Widget API</a> - a Javascript API for creating Portals dashboard widgets</li> <li><a href="portals/">Portals API</a> - an API for interacting with the Portals application (beta)</li> </ul> <h3 id="external-links">External Links</h3> <ul> <li><a href="https://support.exosite.com/hc/en-us/sections/200072527">One Platform Reference</a></li> <li><a href="https://support.exosite.com/hc/en-us/sections/200072708">Portals Web Application Reference</a></li> <li><a href="https://support.exosite.com/hc/en-us/sections/200054894">Portals Subdomain (Whitelabel) Administration Reference</a></li> <li><a href="https://github.com/exosite-labs">Exosite Libraries</a> - Python, Java, .NET, Cocoa, C++, ...</li> </ul> <h2 id="contribution">Contribution</h2> <p>Please tell us how we can make our documentation better. If you have a specific feature request or if you noticed something missing, please let us know by creating a <a href="https://github.com/exosite/api/issues">GitHub issue</a>, or fork the Github project and send a pull request with improvements. Thanks!</p>
